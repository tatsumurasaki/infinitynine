(စာသင်ခန်းသုံး ဘာသာစကား) (Jiàoshì yòng yǔ) (Language used in the classroom)
No. Burmese Chinese English
1. မင်္ဂလာပါ၊ အားလုံး မင်္ဂလာပါ၊ ဆရာ/မ မင်္ဂလာပါ။ nǐ hǎo! dàjiā hǎo! lǎoshī hǎo! (您好! 大家好! 老师好!) hello! hello everybody! hello sir/ma'am!
2. အားလုံး မင်္ဂလာ နံနက်ခင်းပါ။ dàjiā zǎoshang hǎo! (大家早上好!) Good morning, everybody!
3. အားလုံး မင်္ဂလာ နေ့လယ်ခင်းပါ။ dàjiā zhōngwǔ hǎo! (大家中午好!) Good afternoon, everybody!
4. အားလုံး မင်္ဂလာ ညချမ်းပါ။ dàjiā wǎnshang hǎo! (大家晚上好!) Good evening, everybody!
5. ရလား? ရတယ်/ မရဘူး။ kěyǐ ma? kěyǐ / bù kěyǐ (可以吗? 可以 / 不可以) can? can / can't
6. ပြီးပြီလား? ပြီးပြီ / မပြီးသေးဘူး hǎole ma? hǎole / hái méi hǎo(好了吗? 好了 / 还没好) ok? alright / not yet
7. ဖတ်တတ်လား? ဖတ်တတ်တယ်/ မဖတ်တတ်ဘူး။ huì dú ma? huì dú / bù huì dú(会读吗？会读 / 不会读) can you read? can read / can't read
8. ရေးတတ်လား? ရေးတတ်တယ်/ မရေးတတ်ဘူး။ huì xiě ma? huì xiě / bù huì xiě(会写吗？会写 / 不会写) can you write? can write / can't write
9. လွယ်လား? လွယ်တယ်/ မလွယ်ဘူး။ róngyì ma? róngyì / bù róngyì (容易吗？容易 / 不容易) is it easy? easy / not easy
10. ခက်လား? ခက်တယ်/ မခက်ဘူး။ nán ma? nán / bù nán (难吗? 难 / 不难) is it difficult? difficult / not difficult
11. ကျေးဇူးပြု၍ နောက်တစ်/နှစ်ခေါက် ထပ်ဖတ်ပါ။ qǐng zài dú yí(liǎng) biàn (请再读一(两)遍)
12. ကျေးဇူးပြု၍ နောက်တစ်/နှစ်ခေါက် ထပ်ပြောပါ။ qǐng zài shuō yì(liǎng) biàn (请再说一(两)遍) please say one/two more times
13. ကျေးဇူးပြု၍ နောက်တစ်/နှစ်ခေါက် ထပ်နားထောင်ပါ။ qǐng zài tīng yí(liǎng) biàn (请再听一(两)遍) please listen one/two more times
14. ပြဿနာရှိလား (မေးစရာရှိလား)? ရှိတယ်/ မရှိဘူး။ yǒu wèntí ma? yǒu / méiyǒu(有问题吗? 有 / 没有) Is there a problem? Yes / No
15. နားထောင်လို့ နားလည်လား? နားထောင်လို့ နားလည်တယ်၊ နားထောင်လို့ နားမလည်ဘူး။ tīng dǒng ma? tīng dǒng / tīng bù dǒng (听懂吗？听懂 / 听不懂) understand? understand/ I don't understand
16. အားလုံး နောက်မှတွေ့မယ် (Goodbye!)။ dàjiā, zàijiàn! (大家再见!) everybody goodbye!
17. ဆရာ/မ Goodbye! ဆရာ/မ ကျေးဇူးတင်ပါတယ်။ ဆရာ/မ ပင်ပန်းသွားပါပြီ။ lǎoshī, zàijiàn! xièxie lǎoshī, lǎoshī xīnkǔ le (老师, 再见! 谢谢老师, 老师辛苦了) Goodbye, sir/ma'am! Thank you, sir/ma'am, sir/ma'am worked hard
18. နင့်ကို ကျေးဇူးတင်ပါတယ်/ အားနာစရာ မလိုပါဘူး။ xièxie nǐ / bú kèqì (谢谢你 / 不客气) thank you/ you are welcome
19. တောင်းပန်ပါတယ်/ကိစ္စမရှိပါဘူး။ duìbuqǐ / méi guānxi (对不起 / 没关系) sorry / it doesn't matter
20. ကျေးဇူးပြု၍ လက်ထောင်ပါ။ qǐng jǔ shǒu. (请举手。) please raise your hands.
21. ကျေးဇူးပြု၍ တိတ်တိတ်နေပါ။ qǐng ānjìng. (请安静。) please be quiet.
